356kph

A Web Controlled Arduino RC Car

Download this project as a .zip file Download this project as a tar.gz file

Overview

Turn any old RC car into a Wi-Fi enabled rover in a few simple steps. Then use any web enabled device to control your creation.

Hardware

You'll need a simple 4-function RC car (forward, reverse, right, left) and an Arduino compatible Wi-Fi board.

Software

Simply download the Arduino sketch, upload it to your board, and then visit this page when you're ready.

Make

Ready to make it yourself? Head on over to the wikito get started.
